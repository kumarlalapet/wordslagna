So the only paths are:

C
CJ

C → aux
CJ → aux


C → V
CJ → V

C  → V → aux 
CJ → V → aux 


IV

C = Consonant
CJ = Common Conjuncts
Aux = Auxiliary (4 chnadrabindu)
V = non independent vowels
IV = independent vowels
